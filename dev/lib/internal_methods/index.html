HighDimMixedModels.special_quad — Method

Calculates (y-ỹ)'(invV)X[:,j], where ỹ are the fitted values if we ignored the jth column i.e. XG[:,Not(j)]β[Not(j)] To improve perforamce, we calculate ỹ with the entire dataset We then split into groups and calculate (y-ỹ)'(invV)*X[:,j] for each group
